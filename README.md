# 🖥️ Computer Club Management System

**Computer Club** — это система для управления компьютерным клубом. Приложение позволяет вести учет клиентов, компьютеров, бронирований и платежей, а также управлять пользователями системы.

---

## 📋 Функционал

* Управление клиентами: добавление, редактирование, удаление информации о клиентах.
* Управление компьютерами: учет всех компьютеров в клубе, их состояния и местоположения.
* Бронирование: возможность бронировать компьютеры на определенное время.
* Управление платежами: учет всех платежей клиентов за аренду компьютеров.
* Авторизация пользователей: разные уровни доступа для администраторов и сотрудников.
* Отображение данных в удобном графическом интерфейсе.

---

## 🛠 Технологии

* **Python** – основной язык разработки
* **Tkinter** – графический интерфейс
* **SQLite/MySQL/PostgreSQL** – база данных (в зависимости от настроек)
* **Python OOP** – объектно-ориентированное программирование
* **SQLAlchemy / sqlite3** – для работы с базой данных
* **Python 3.10+**

## ⚙️ Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/username/computer_club.git
cd computer_club
```

2. Создайте виртуальное окружение и активируйте его:

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Настройте базу данных в `db/connection.py` (SQLite по умолчанию).

5. Заполните тестовыми данными (опционально):

```bash
python seed_data.py
```

6. Запустите приложение:

```bash
python ui/app.py
```

## 🔒 Авторизация

Система поддерживает несколько уровней пользователей:

* **Администратор** – полный доступ к данным и настройкам.
* **Сотрудник** – ограниченный доступ, может только работать с бронированиями и платежами.
